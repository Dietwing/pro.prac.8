
# Отчет по проектному практикуму. Задание 8

В рамках работы рассматривается разработка программного приложения для поддержки настольно-ролевых игр.
Основная задача приложения заключается в автоматизации вычислений игровых параметров персонажей,
организации хранения информации о персонажах, а также упрощении работы гейм-мастера во время игровых сессий.

Функциональные возможности системы включают:
- создание и редактирование листов персонажей
- хранение характеристик персонажей
- управление боевыми событиями
- выполнение случайных бросков кубов
- ведение заметок игровой сессии

Хранение данных осуществляется локально с использованием файлов JSON и базы SQLite.
